# Software_Project
Project Name: Personalized Workout Planner & Progress Tracker

Overview:
A web app that generates customized workout plans based on user goals (e.g., weight loss, muscle gain) and tracks progress via logged workouts, with adaptive adjustments over time.

Problem it Solves:
Users struggle with unstructured fitness routines; this provides personalized plans, progress visualization, and motivation through data-driven insights.

Target Users & Personas:
Beginner Gym-Goer (Priya, 25, office worker): Wants simple weight loss plans, tracks calories/workouts on mobile.
Fitness Enthusiast (Alex, 30, athlete): Seeks muscle gain programs with progress analytics and adjustments.
Busy Professional (Raj, 35): Needs quick strength sessions with minimal setup.

Vision Statement:
Empower anyone to achieve fitness goals with AI-driven, adaptive plans and seamless tracking.

Key Features:
Goal-based workout generation.
Daily workout logging.
Progress charts and adjustments.
User profiles and history.

Goals & Success Metrics:
80% user retention after 30 days.
Average 4 workouts logged/week.
Plans adjusted based on 90% adherence.

Assumptions & Constraints:
Users have basic web access; no hardware sensors; built with open-source stack; Agile sprints of 2 weeks.
